So little time - Jan 21, 2017

I've been researching a little more about Web Development and what it takes to be a 'good' Web Developer.
But with any type of Computer Programming, you have to realise your skill set and
what you enjoy doing most or what aspects of programming you're good at.
So let's break it down.

In programming, there's 3 types of developers.
There are Front-End Developers, Back-End Developers and the godly Full-Stack Developers

Front-End Developers -


Source: favicondigital.com

These types of developers are responsible for making the website or product 'look pretty'.
Anything from fonts, color combinations, medias, UI, UX and everything else, they're the people behind it.
They work closely with designers to create ideas of marrying a clean interface with a functional one.

To be honest, it's a lot easier said than done.

I've seen wayyy too many 'one page websites' or 'uber xtreme LEL ALL THE medias play all at once NGNGNGNGNNG' websites thinking that it will run as well online as it did locally.
Like, no dude.

For the people who don't have gigabit internet, its going to look choppy as hell.
Which makes me want to leave.
So it's more complicated than it sounds.

Back-End Developers -


Source: favicondigital.com

Back-End developers are responsible for the data that gets transmitted and stored from and to the website.
They are responsible for managing the back-end of the website.
These consist of a server, an application and a database.
These three aspects are what give the website and pages life and content.

The languages associated with being a Back-End developer consists of server-side languages such as
PHP, Ruby, Python, Java and .NET for the application.
As well as tools like MySQL, Oracle and SQL Server to find/save/change data and
serve it back to the user in front-end code.
There are a whole host of other frameworks and control software like Git that
back-end developers need to be able to master.

All of this in combination, is the reason why users can navigate around
websites and see content on their screen.
The reason why you log into your account on a website or program or app and it loads all of your saved files is all because of the work of Back-End Developers.

And finally,

Full-Stack Developers -


Source: sitepoint.com

Full-Stack developers are insane.
They are literally a one-person team.
They have both the skills of a front-end developer and a back-end developer.
They are able to create and stabalise the back-end technologies of the product and also create and design a aesthetically pleasing product whilst also being able to deliver a satisfying experience, and then some.
But to be honest, nowadays, it isn't that hard to be somewhat of a Full-Stack developer.
Hell, I created this entire website from scratch, designed it, enabled it on a hosting platform.
So ... I'm technically a Full-Stack developer.
(not really)

But yeah, that's pretty much the 3 types of developers.
I wish I had more free time to play around with some more Front-End and also Back-End programming languages and implement them into my personal projects.
I'm slowly starting to realise its almost the end of my summer break.

There never seems to be enough time.
